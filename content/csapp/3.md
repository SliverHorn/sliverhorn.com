## Memory Reference

$$
Imm(r_b,r_i,s) => Imm + R[r_b]+R[r_i]{\cdot}s
$$

|       |                 |       |
| ----- | --------------- | ----- |
| Imm   | Immediate       | 立即数   |
| $r_b$ | Base Redgister  | 基址寄存器 |
| $r_i$ | Index Redgister | 变址寄存器 |
| $s$   | Scale Factor    | 比例因子  |
